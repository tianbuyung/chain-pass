# Trust Pass

## Description

This application would be a ticketing system app using blockchain.

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run dev

# build
$ pnpm run build

# production mode
$ pnpm run start
```
